# Horoscope

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.1.1.

It has been built with Angular plus the use of Bootstrap. A dynamic fetch API request is executed each time the user selects a different sign and day.

Apart from it, the web app has been coded by using a Guard, Resolve, Services, RxJs, Local Storage, Lazy Loading and Interfaces. The architectural pattern for the Horoscope App is MVVM.

## Firebase deploy

You can access the deployed web app at `https://horoscope-3ae84.web.app/`
